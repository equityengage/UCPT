== Description ==
This plug-in supports the Universal Community Planning Tool (UCPT) framework.

= UCPT Configuration =

Please note that all Universal Community Planning Tool (UCPT) components are powered by an options panel named UCPT Manager Plug-In. This plug-in is required to run any of the UCPT template modules.

= Support =

Support is available on https://equityengage.com, and via the plug-in page on WordPress.org.

= Installation =
1. Go to Add New Plug-In
2. Search UCPT
3. Install and Activate the UCPT Manager Module
4. Install and Activate Optional UCPT Driver Modules, such as the UCPT Strategy Card Module or UCPT Raw Data + Module

All settings are managed through the UCPT Manager Module plug-in.
